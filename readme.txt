A BuddyPress addon for exporting and importing user data with field mapping and WP CLI support.

== Description ==
This addon provides comprehensive tools for exporting and importing BuddyPress user data, including field mapping and WP CLI commands.

== Installation ==
1. Upload `bp-export-import` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
